# grid_converter_pipeline

Automated pipeline for PowerFactory, Python, and MATPOWER.

## Overview

This project focuses on the modelling and analysis of power systems with increasing penetration of converter-based generation. The main objective is to automate the workflow between DIgSILENT PowerFactory, Python, and MATPOWER, enabling efficient creation and evaluation of different power system scenarios.

## Project Objectives

- Model electrical power systems in DIgSILENT PowerFactory.
- Replace conventional synchronous generation with converter-based renewable generation.
- Extract network data automatically using Python.
- Convert PowerFactory models into MATPOWER-compatible cases.
- Analyse the impact of converter penetration on power system behaviour.
- Reduce manual work through an automated and reproducible workflow.
